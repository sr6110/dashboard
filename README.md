## Setup

1. Clone the repository:
   ```bash
   git clone <need to add url>
   ```
2. Install dependencies:
   ```bash
   cd dashboard
    npm install
    ```

3. Start the development server:
   ```bash
    npm run dev
    ```
4. Access the dashboard in your browser, typically at http://localhost:5173/.
